# Changelog

All notable changes to the FHNW Presentation Template will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-12-02

### Added
- Initial release of FHNW Presentation Template for Typst
- Professional presentation theme following FHNW branding guidelines
- Official FHNW colors (yellow, dark gray, red, teal)
- Multiple slide layouts: regular, section dividers, two-column, three-column
- Content blocks: example, definition, alert, and code blocks
- Automatic page numbering with slide counter
- Configurable presentation parameters (title, authors, date, logo)
- Utility functions: `section-slide`, `two-columns`, `three-columns`
- Styled blocks: `example-block`, `definition-block`, `alert-block`, `code-block`
- Special slides: `last-slide` for closing slides, `cite-slide` for references
- Support for math equations, tables, and figures
- Professional typography with font fallbacks (Roboto → Liberation Sans)
- Monospace fonts for code blocks (Source Code Pro → Liberation Mono)
- Customizable Swiss Universities logo in title slide
- Complete example presentation demonstrating all features

### Features
- 16:9 aspect ratio support
- Optional subtitle and email display
- Configurable institute name
- Page numbering (current/total)
- Header and footer styling
- Dashed section dividers
- Bullet and numbered list support
- Text styling: bold, emphasis, colored text, links
- Professional code block styling
